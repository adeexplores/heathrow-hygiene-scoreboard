FSA RATING BADGE IMAGES
=======================

The official Food Standards Agency rating badges live in this folder, named by
the FSA "rating key" so the widget and the review-page badge can find them:

    fhrs_0_en-gb.png   ... rating 0
    fhrs_1_en-gb.png   ... rating 1
    fhrs_2_en-gb.png   ... rating 2
    fhrs_3_en-gb.png   ... rating 3
    fhrs_4_en-gb.png   ... rating 4
    fhrs_5_en-gb.png   ... rating 5   (currently the official "Design version 3" badges)

WHAT'S NOT HERE (and that's fine)
    Some venues have a status instead of a number, e.g. "AwaitingInspection" or
    "Exempt". There is no numbered badge for those, so the widget/badge shows the
    wording in a small grey chip instead. No image file is needed for them.

IF YOU WANT TO SWAP THE BADGE STYLE
    Download another official set from the FSA (search "FHRS displaying your
    rating" on gov.uk), and replace the files above keeping the SAME file names.

RULES (from the FSA terms - ratings.food.gov.uk/terms-and-conditions)
    - Use the official images only; do not alter or recolour them.
    - Only ever show a venue's CURRENT rating (this project does, because it
      reads live FSA data).
    - Showing the rating is not an FSA endorsement - the widgets say so.
